## Project Description

This project is a Hospital Management System (HMS) developed as part of the Database Systems course at Addis Ababa Science and Technology University (AASTU).

The system is designed to manage and organize hospital operations efficiently by handling patient records, doctor information, appointments, medical records, billing, payments, emergency cases, and queue management.

The project applies both relational and NoSQL database concepts using MySQL and MongoDB. It also demonstrates database analysis, ER modeling, normalization up to BCNF, schema implementation, querying, and testing.

##  How to Run the Project

### MySQL

1. Open MySQL Workbench or MySQL CLI.
2. Import or run:

   * `mysql/schema.sql`
   * `mysql/queries.sql`
3. Execute the queries to create tables and test the database.

### MongoDB

1. Open MongoDB Compass or Mongo Shell.
2. Import:

   * `mongodb/collections.json`
3. Run:

   * `mongodb/queries.js`
4. Execute the queries for testing and data retrieval.
